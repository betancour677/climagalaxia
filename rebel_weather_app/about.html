Sobre el Proyecto

Rebel Weather es una aplicación de clima diseñada para la Alianza Rebelde. Es un proyecto desarrollado para el curso de Frontend, aplicando tecnologías modernas de la web.

Tecnologías Utilizadas:
  • • HTML5 Semántico
  • • CSS3 Personalizado
  • • Bootstrap 5 (Grid, Layout, Components)
  • • JavaScript (Manipulación del DOM, API Simulada)
  • • Git & GitHub (Control de Versiones)
Propósito

Este MVP sirve para demostrar la integración de estilos responsivos con lógica de navegación dinámica en una aplicación de una sola página (SPA-like) con múltiples vistas.
